# 项目说明
比较简洁的vue3+ts+naive-ui+unocss的后台管理系统模板,适合再次开发。项目只有简单的页面，没有复杂的业务逻辑，只是一个简单的模板，适合新手学习。


- 待添加

```bash
# 安装依赖
pnpm i
# 启动项目
pnpm run serve
```
